# Food Waste Prediction System

## Overview
Machine Learning-based system to predict food wastage for
hostels, hotels, weddings, and households.

## Tech Stack
- Python
- Flask
- Scikit-learn
- HTML/CSS

## Project Structure
- App/        → Flask app
- ML/         → Model training scripts
- models/     → (excluded, downloaded separately)
- data/       → (sample data only)
